# Data Intelligence PRO

최첨단 알고리즘 기반의 데이터 통합 및 추출 솔루션입니다. Premium UI와 전문가용 옵션을 통해 복잡한 데이터 작업을 손쉽게 처리할 수 있습니다.

## 주요 기능
- **🔗 데이터 매칭**: 두 데이터를 키 기준으로 병합 (VLOOKUP 방식)
- **📄 단일 파일 추출**: 정밀 필터링 및 중복 제거 추출
- **📊 열려있는 엑셀 연동**: 작업 중인 엑셀 시트 즉시 파싱
- **📂 데이터 병합**: 여러 파일을 하나의 통합본으로 결합
- **📈 데이터 분석**: 통계 및 빈도 자동 리포팅

## 기술 스택
- **Language**: Python 3.9+
- **UI Framework**: PySide6 (Qt for Python)
- **Data Engine**: Pandas, OpenPyXL
- **Windows Integration**: pywin32 (for Excel COM)

## 시작하기
1. 의존성 설치: `pip install -r requirements.txt`
2. 실행: `python app/main.py` 또는 `./run.sh`
